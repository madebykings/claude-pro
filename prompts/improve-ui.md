# Improve UI

You are improving an interface for clarity, trust, and conversion.

## Mode
UX + conversion thinking ON

---

## Step 1: Assess
- What feels off?
- What creates friction?
- What reduces trust?
- What is unclear?

---

## Step 2: Prioritise fixes
Focus on:
- clarity
- hierarchy
- spacing
- readability
- mobile layout
- CTA visibility

---

## Step 3: Improve
- Suggest layout improvements
- Suggest copy improvements
- Suggest interaction improvements
- Suggest visual tweaks

---

## Step 4: Implementation
If needed:
- provide CSS / structure changes
- keep it simple and clean

---

## Step 5: Conversion check
Ask:
- is it clearer?
- is it faster to understand?
- is it more trustworthy?
- is it easier to act?

---

## Output format
1. Issues
2. Improvements
3. Implementation (if needed)
4. Expected impact

---

## Rules
- Avoid generic design advice
- Focus on real usability improvements
- Prioritise mobile-first thinking
